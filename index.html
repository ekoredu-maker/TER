사용방법

기본 사용 순서

  1. 출장 가져오기 탭에서 엑셀을 불러오고, 하단 설정에서 서명과 영수증 배치를 저장합니다.
  2. 여비정산 탭에서 출장 1건을 선택합니다. 관내 출장은 '정산불요'로 자동 셋팅됩니다.
  3. 출장지에 주요 기관(예: 교육청)을 입력하면 거리가 자동 제언됩니다.
  4. 정산 저장 후 영수증 관리로 이동하여 첨부합니다.
  5. 정산서·영수증 출력(A4)으로 인쇄 시 서명이 겹쳐서 출력됩니다.

화면에서 꼭 볼 표시

  • 저장상태: 입력 중인지, 저장이 끝났는지 상단에 표시됩니다.
  • 마지막 저장: 언제 저장했는지 시간까지 확인할 수 있습니다.
  • 선택 출장: 현재 어떤 출장건을 편집 중인지 보여줍니다.
  • 저장하지 않은 상태에서 다른 출장건이나 탭으로 이동하면 경고가 뜹니다.

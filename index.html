사용방법 · v1.1 서식정밀화판

기본 사용 순서

  1. 출장 가져오기 탭에서 출장 목록 그리드 엑셀을 불러옵니다.
  2. 여비정산 탭에서 출장 1건을 선택하고 소속·기간·경비를 확인합니다.
  3. 이동구분에서 자가용 또는 대중교통을 먼저 고르고, 그다음 왕복/편도를 선택합니다. 왕복이면 자동으로 2행이 생성됩니다.
  4. 정산 저장 후 영수증 관리로 이동하여 이미지 또는 PDF를 첨부합니다.
  5. 정산서·영수증 출력(A4)으로 제출용 인쇄물을 만듭니다.
  6. GitHub Pages에 올린 뒤에는 앱 설치 버튼이나 브라우저 메뉴를 이용해 PWA처럼 바로가기 설치가 가능합니다.

화면에서 꼭 볼 표시

  • 저장상태: 입력 중인지, 저장이 끝났는지 상단에 표시됩니다.
  • 마지막 저장: 언제 저장했는지 시간까지 확인할 수 있습니다.
  • 선택 출장: 현재 어떤 출장건을 편집 중인지 보여줍니다.
  • 저장하지 않은 상태에서 다른 출장건이나 다른 탭으로 이동하면 경고가 뜹니다.

인쇄 전 점검

  • 용지는 A4, 방향은 세로로 둡니다.
  • 자가용 경로를 썼다면 출발지·도착지가 비어 있지 않은지 확인합니다.
  • 영수증은 이미지와 PDF 모두 가능하지만, 큰 PDF는 저장 시간이 조금 더 걸릴 수 있고, 브라우저 특성상 인쇄 결과가 다를 수 있습니다.
  • 출력 전에 한 번 더 정산 저장을 눌러 최신 입력값을 반영하는 것이 안전합니다.

자주 있는 문제

  • 입력값이 안 보일 때: 저장 후 다시 미리보기를 열어 보세요.
  • 다른 PC로 옮길 때: 백업/복원 탭에서 JSON 백업 파일을 함께 보관하세요.
  • 영수증이 안 보일 때: 해당 정산서를 먼저 선택했는지 확인하세요.
  • 인쇄가 맞지 않을 때: 브라우저 인쇄창에서 A4와 기본 배율을 확인하세요.
